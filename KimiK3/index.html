中国开源模型，
第一次坐进前沿牌桌

2.8 万亿参数、100 万 token 上下文、独立评测全球第 4——Kimi K3 让中国模型首次追平西方上一代旗舰（Claude Opus 4.8 / GPT-5.5），并咬住最新一代（Fable 5 / GPT-5.6 Sol）。同时它把价格提到接近西方中端模型，宣告"中国 AI = 超低价"时代进入尾声。

本报告的一句话判断：K3 是能力上的里程碑、商业上的豪赌——"开源"成色等 7 月 27 日权重落地才能验收，涨价能否站住等独立复测和市场投票。 核心结论 · 五路调研 + 12 条关键结论逐条核验

KEY FINDINGS · 核心发现

五条带走的结论

01 · 位置

追平上一代，咬住最新一代

Artificial Analysis 独立评测 57.1 分、全球第 4：压过 Claude Opus 4.8，距 Claude Fable 5（59.9）一档之遥；LMArena 前端编程盲测甚至排第一。对开源阵营则断档领先约 6 分。→ 竞争格局

02 · 技术

不是放大，是换引擎

KDA 混合线性注意力（1M 上下文最高 6.3× 解码加速）+ AttnRes + Gated MLA + Stable LatentMoE（896 选 16），官方称训练效率较 K2 提升 2.5 倍。技术报告未发，数字待复现。→ 技术拆解

03 · 定价

主动告别超低价

$3 输入 / $15 输出，数倍于自家上代、5-10 倍于 DeepSeek——被解读为"中国 AI 超低价时代终结"的第一个信号。是自信还是自断武器，取决于能力差距能保持多久。→ 竞争格局

04 · 节奏

2-3 个月一发版的恐怖迭代

K1.5 → K2 → K2 Thinking → K2.5 → K2.6 → K2.7-Code → K3，十八个月七次实质发版；估值同步从 43 亿美元冲到 200 亿+，ARR 六周翻倍。→ 时间线

05 · 反面

掌声之下，账没有结清

C 端月活从峰值跌去六到七成（主动弃 C 保 B）；约 100 倍 P/ARR 的估值倍数；发布首日"慢且贵"的实测；"开源"权重发布日为空档；仲裁旧案未结、Anthropic 蒸馏指控未回应。这些不推翻 K3 的技术成就，但它们决定这场豪赌的另一半胜负。→ 争议与反方

行业坐标

开源追平前沿的时刻

百人团队的极限压强
