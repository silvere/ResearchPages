# ResearchPages · 调研台

每一个文件夹是一份深度调研档案。接到一个主题后，按「扇出调研 → 事实核验 → 建站成档 → 上架归档」四阶段闭环，产出一座零依赖、可交互的静态专题站点，部署于 GitHub Pages。

规范与流程见 [AGENTS.md](AGENTS.md)。本地预览：`python3 -m http.server 8000`。
